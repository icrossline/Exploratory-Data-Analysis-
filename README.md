A comprehensive collection of SQL scripts for data exploration. It helps to gather more information from the data and can  be used for business growth etc,. 
These scripts cover various analyses such as-:
      1- **DATABASE EXPLORATION** : To explore the structure of Database .Inspect the columns and metadata for specific table.
      2- **DIMENSION EXPLORATION** :  To explore the structure of tables .To Gather information like  Unique , Distinct values  etc,.
      3- **DATA RANGE EXPLORATION** : To explore the History range of data.
      4- **MEASURES** : Use aggregate function to calculate value for quick insights like Total_revenue , Total_items_sold etc,.
      5- **MAGNITUDE ANALYSIS** : - Compare  measure value for each category.
      6 -**RANKING ANALYSIS** : Rank Dimensions  by measure like top 5 sold product.

DATASET-Contains the file in which you can perform EDA .
DOCS - Contain the project_roadmap.
